Summary

Position:  GenAI Engineer

Location: Remote within the East Coast time zone

Duration: 6+ Months

 

Required Experience

Cloud & Infrastructure

·       4+ years on Google Cloud Platform in production, with hands-on experience across:

·       Cloud Run (containerized service deployment)

·       Firestore (NoSQL document data modeling)

·       BigQuery (data warehousing, SQL, performance tuning)

·       Strong CI/CD experience — pipeline design, build automation, deployment gates

·       Git proficiency, specifically with Azure DevOps (Repos + Pipelines)

Back End

·       5+ years of professional Python development

·       Production experience with FastAPI (REST APIs, async, dependency injection)

·       Strong working knowledge of Pydantic (v1 or v2) for schema validation and settings

Front End

·       3+ years building production applications with Next.js

·       Solid JavaScript / TypeScript fundamentals

·       Comfort building responsive, component-based UIs

General

·       8+ years total professional software engineering experience

·       Demonstrated ability to operate as a senior full-stack engineer — owning features end-to-end without hand-offs

·       Strong written and verbal English communication; comfortable in client-facing standups

·       US-based, authorized to work in the US, and able to work core East Coast hours

 

Preferred / Bonus

·       Prior healthcare, payer, or HIPAA-regulated environment experience

·       Prior work on GenAI / LLM / RAG systems — vector databases, embedding pipelines, prompt orchestration, agent frameworks

·       Prior exposure healthcare enterprises

·       Experience with data source integration at scale (APIs, ETL, batch + streaming)

·       Familiarity with observability tooling on Google Cloud Platform (Cloud Logging, Cloud Monitoring)

·       Experience working in distributed contractor pods alongside enterprise client teams
